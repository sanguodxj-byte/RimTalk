# 快速开始指南

## 第一步：配置RimWorld路径

在构建之前，你需要修改项目文件中的RimWorld引用路径。

### Windows (Steam)
编辑 `Source/LivingWeapons/LivingWeapons.csproj`，将以下路径：

```xml
C:\Program Files (x86)\Steam\steamapps\common\RimWorld\RimWorldWin64_Data\Managed\
```

修改为你的实际RimWorld安装路径。

### 如何找到你的RimWorld路径？

1. 打开Steam
2. 右键点击RimWorld -> 管理 -> 浏览本地文件
3. 复制路径，添加 `\RimWorldWin64_Data\Managed\`

## 第二步：构建模组

### 方法1：使用PowerShell脚本（推荐）

```powershell
.\Build.ps1
```

### 方法2：手动构建

```powershell
cd Source\LivingWeapons
dotnet build -c Release
```

## 第三步：安装模组

将整个模组文件夹复制到：

```
[RimWorld安装目录]\Mods\LivingWeapons\
```

例如：
```
C:\Program Files (x86)\Steam\steamapps\common\RimWorld\Mods\LivingWeapons\
```

## 第四步：启动游戏

1. 启动RimWorld
2. 进入模组管理界面
3. 找到并启用 "Living Weapons - 活体兵器"
4. 重启游戏

## 开始游戏！

### 如何获得活体兵器？

1. **制作垃圾武器**
   - 让你的工匠制作武器
   - 品质越差，成为活体兵器的概率越高！
   - 极差品质：20%概率
   - 差品质：10%概率
   - 普通品质：1%概率

2. **击败敌人**
   - 敌人有10%概率携带已觉醒的活体兵器
   - 等级在1-5之间

3. **购买或交易**
   - 从商人那里购买的低品质武器也可能是活体兵器

### 如何使用活体兵器？

1. **查看状态**
   - 选中武器，查看详细信息
   - 潜伏期会显示经验进度
   - 觉醒后会显示等级和属性

2. **培养成长**
   - 让殖民者装备武器战斗
   - 击杀敌人获得50经验
   - 造成伤害获得少量经验
   - 100经验觉醒，之后每级需要更多经验

3. **避免饥饿**
   - 每3天必须让武器见血
   - 否则会开始降级
   - 可以利用这个机制重铸属性

### 游戏技巧

1. **废土捡漏流**
   - 专门制作大量极差武器
   - 筛选出活体兵器进行培养
   - 其他的熔炼或出售

2. **重铸流**
   - 养到Lv.5发现属性不好
   - 故意饿几天降级
   - 重新练级赌好属性

3. **敌人狩猎流**
   - 击杀强敌获取高等级武器
   - 跳过前期培养直接使用

## 常见问题

**Q: 为什么我的武器没有变化？**
A: 活体兵器在潜伏期外观完全正常，需要积累经验后觉醒。

**Q: 如何知道敌人的武器是否已觉醒？**
A: 查看敌人装备的详细信息，会显示等级。

**Q: 高科技武器经验获取太慢？**
A: 这是设计如此，防止批量工业生产。建议使用低科技武器。

**Q: 武器降级了怎么办？**
A: 立即让它参加战斗，或者利用降级重铸更好的属性。
