## ??? 开发工具（Dev模式）

### 使用方法
1. 按 `F11` 开启Dev模式
2. 按 `~` 打开Debug Actions
3. 搜索 "Living Weapons"

### 三大工具
```
1. 生成极差武器    → 快速测试
2. 测试概率(100次) → 验证20%
3. 生成Lv.5武器    → 跳过培养
```

## ?? 安装速记

```powershell
# 复制到Mods
Copy-Item -Path . -Destination "D:\steam\steamapps\common\RimWorld\Mods\LivingWeapons\" -Recurse -Force

```
