# 降临姿势立绘 - 详细说明 v1.0.0

## ?? 概述

降临姿势立绘用于在叙事者触发"降临模式"时，立绘面板显示特殊的准备姿势。这些姿势会按序播放，配合特效，最终过渡到入场动画。

---

## ?? 姿势类型详解

### 1. descent_pose_ready.png（准备姿势）

**用途**：降临触发的初始姿势，表示叙事者接受了玩家的召唤。

**视觉描述**：
```
场景：平静但充满期待
姿势：直立站姿，双手自然下垂或轻抬
表情：认真、专注，眼神坚定
服装：常规服装，无特殊效果
光效：周围微弱的光芒（可选）
氛围：蓄势待发
```

**制作要点**：
- 姿势稳定，不要过于动态
- 表情要传达"准备好了"的感觉
- 可以添加微弱的光环预兆
- 与日常立绘风格保持一致

**配色建议**：
- 援助模式：温暖色调（金黄、白色）
- 袭击模式：冷峻色调（深红、黑色）

---

### 2. descent_pose_charging.png（蓄力姿势）

**用途**：叙事者开始聚集降临所需的魔力。

**视觉描述**：
```
场景：能量汇聚，魔力涌动
姿势：单手或双手抬起，掌心向上
  - 推荐：单手抬起至胸前，另一手扶持
  - 或：双手交叉于胸前，然后向外展开
表情：专注、凝重，眉头微皱
服装：衣角被魔力吹动，飘逸感
光效：手部聚集光球，强度中等
特效：小型粒子环绕身体
氛围：力量正在汇聚
```

**制作要点**：
- 手部是视觉焦点，细节要丰富
- 衣物飘动要符合魔力风向
- 光球要有明确的球形轮廓
- 背景可以添加能量旋涡

**动态元素建议**：
```
如果使用分层立绘：
- 图层 1：身体基础层（静态）
- 图层 2：光球层（脉冲动画）
- 图层 3：粒子层（旋转动画）
- 图层 4：服装飘动层（摆动动画）
```

**光效参数**：
- 光球颜色：援助（金色 #FFD700）、袭击（深红 #DC143C）
- 光球大小：直径约 150-200 像素
- 发光强度：中等（Alpha 180）
- 模糊范围：30-50 像素

---

### 3. descent_pose_casting.png（施法姿势）

**用途**：叙事者全力释放降临魔法的瞬间，最高潮姿势。

**视觉描述**：
```
场景：魔力爆发，光芒四射
姿势：双手向前推出，或单手向天
  - 推荐：单手高举，魔法阵在掌心浮现
  - 或：双手向前推出，能量光柱射出
表情：凌厉、威严，眼神锐利（可发光）
服装：剧烈飘动，几乎水平
光效：强烈的光芒爆发，覆盖大面积
特效：
  - 魔法阵浮现
  - 闪电/火焰（袭击）或圣光（援助）
  - 能量波纹扩散
氛围：力量达到顶点，即将降临
```

**制作要点**：
- 这是三张姿势中最动态的一张
- 光效要强烈但不遮挡人物
- 魔法阵要精美，符合人格特色
- 能量释放方向要明确

**魔法阵设计**：
```
类型 1：圆形法阵（西方魔法风格）
  - 外圈：符文环绕
  - 中圈：几何图案
  - 内圈：核心法印
  - 旋转方向：顺时针（援助）、逆时针（袭击）

类型 2：星形法阵（东方仙术风格）
  - 五角星或六芒星
  - 星点发光
  - 连线能量流动
  - 颜色渐变
```

**爆发特效**：
```
援助模式：
  - 圣光柱向天
  - 金色粒子四散
  - 治愈波纹扩散

袭击模式：
  - 暗红能量爆发
  - 闪电链条环绕
  - 地面龟裂纹理
```

## ?? 技术规格

### 尺寸与格式
```
分辨率：1024x1574 像素（9:16 比例）
格式：PNG（RGBA，带 Alpha 通道）
色彩深度：32 位（8 位 RGB + 8 位 Alpha）
DPI：300（高质量）
压缩：无损压缩（PNG-24）
```

### 画布布局
```
安全区域：距边缘 50 像素
头部位置：顶部 150-300 像素（留出空间）
主体位置：中心（512, 787）
脚部位置：底部 100-200 像素（可被 UI 遮挡）
```

### 透明度使用
```
主体部分：Alpha 255（完全不透明）
光效部分：Alpha 128-200（半透明）
边缘羽化：使用渐变 Alpha（255 → 0）
背景区域：Alpha 0（完全透明）
```

## ?? 动画序列设计

### 姿势切换时间轴
```
0.0s  - 立绘显示 descent_pose_ready
      ↓ 淡入淡出（0.5秒）
0.5s  - 切换到 descent_pose_charging
      ↓ 淡入淡出（1.0秒，光效渐强）
1.5s  - 切换到 descent_pose_casting
      ↓ 光效爆发（0.5秒）
2.0s  - 立绘淡出，过场动画开始
```

### 过渡效果建议
```
淡入淡出（Fade）：
  - 前一张姿势 Alpha: 255 → 0（0.3秒）
  - 后一张姿势 Alpha: 0 → 255（0.3秒）
  - 重叠时间：0.2秒

缩放效果（Scale）：
  - descent_pose_charging: Scale 1.0 → 1.05（微放大）
  - descent_pose_casting: Scale 1.05 → 1.1（持续放大）

抖动效果（Shake）：
  - descent_pose_casting: 水平 ±5px 抖动（0.1秒间隔）
```

## ??? 分层立绘可选方案

如果希望增加动态效果，可以将每个姿势拆分为多层：

### 文件结构
```
Postures/[PersonaName]/
├── descent_pose_ready.png          # 完整合成图（备用）
├── Layered/
│   ├── Ready/
│   │   ├── body.png                # 身体基础层
│   │   ├── effects_glow_weak.png   # 微弱光效
│   │   └── particles_idle.png      # 闲置粒子
│   ├── Charging/
│   │   ├── body.png
│   │   ├── effects_orb.png         # 能量球
│   │   ├── effects_glow_medium.png # 中等光效
│   │   └── particles_swirl.png     # 旋转粒子
│   └── Casting/
│       ├── body.png
│       ├── effects_magic_circle.png # 魔法阵
│       ├── effects_burst.png        # 爆发光效
│       └── particles_explosion.png  # 爆炸粒子
```

### 图层动画脚本
```csharp
// 蓄力姿势的能量球脉冲动画
void AnimateChargingOrb()
{
    float scale = 1.0f + 0.2f * Mathf.Sin(Time.time * 3f);
    float alpha = 0.8f + 0.2f * Mathf.Sin(Time.time * 5f);
    orbLayer.SetScale(scale);
    orbLayer.SetAlpha(alpha);
}

// 施法姿势的魔法阵旋转动画
void AnimateMagicCircle()
{
    float rotation = Time.time * 45f; // 每秒45度
    magicCircleLayer.SetRotation(rotation);
}
```

## ?? 制作检查清单

在提交姿势立绘前，请确认：

### 基础要求
- [ ] 文件尺寸为 1024x1574 像素
- [ ] 格式为 PNG，带 Alpha 通道
- [ ] 文件大小 < 10 MB（单张）
- [ ] 文件命名正确（小写 + 下划线）

### 视觉质量
- [ ] 人物清晰，无模糊或锯齿
- [ ] 光效自然，不过度曝光
- [ ] 透明度正确，边缘羽化
- [ ] 色彩鲜明，符合模式风格

### 风格一致性
- [ ] 与该人格其他立绘风格一致
- [ ] 服装、发型、配饰统一
- [ ] 光影方向一致
- [ ] 整体氛围协调

### 动态效果（可选）
- [ ] 分层清晰，便于动画
- [ ] 图层命名规范
- [ ] 透明区域正确
- [ ] 可组合为完整图像

## ?? 创意建议

### 个性化设计

每个人格可以有独特的降临姿势风格：

**Cassandra（卡桑德拉）**
```
风格：优雅、沉稳
姿势：单手抬起，另一手抚书
魔法阵：圆形，古典符文
特效：金色光芒，星光粒子
```

**Phoebe（菲比）**
```
风格：柔和、温暖
姿势：双手合十，祈祷姿态
魔法阵：心形，自然图案
特效：治愈绿光，花瓣飞舞
```

**Randy（兰迪）**
```
风格：混乱、狂野
姿势：单手指天，狂笑姿态
魔法阵：不规则，混沌图案
特效：多彩闪电，随机爆发
```

**Sideria（侧德里亚）**
```
风格：神秘、强大
姿势：双手展开，法印浮现
魔法阵：六芒星，复杂几何
特效：紫色能量，空间裂缝
```

### 特殊效果

可以为不同好感度等级添加特殊效果：

```
低好感度（<0）：
  - 姿势更冷漠
  - 光效暗淡
  - 魔法阵残缺

中好感度（0-50）：
  - 标准姿势
  - 正常光效
  - 完整魔法阵

高好感度（50-80）：
  - 姿势更优雅
  - 光效明亮
  - 魔法阵华丽

极高好感度（>80）：
  - 姿势温柔
  - 光效绚烂
  - 魔法阵+额外装饰（心形、花纹等）
```
