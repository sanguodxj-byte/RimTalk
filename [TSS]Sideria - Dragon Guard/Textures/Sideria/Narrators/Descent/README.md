## ?? 美术资源规范

### 1. 降临姿势立绘（Postures/）

#### 用途
- 立绘面板在触发降临前的姿势切换
- 显示叙事者准备降临的状态

#### 规格要求
- **尺寸**: 1024x1574 像素（9:16 比例）
- **格式**: PNG（带 Alpha 透明通道）
- **分辨率**: 300 DPI
- **色彩模式**: RGBA

#### 姿势类型

**1. descent_pose_ready.png（准备姿势）**
```
描述：叙事者站立姿势，双手自然下垂或微微抬起
视觉要素：
  - 平静的表情
  - 光环初现（可选）
  - 轻微的能量波动
```

**2. descent_pose_charging.png（蓄力姿势）**
```
描述：叙事者抬手蓄力，准备施展降临魔法
视觉要素：
  - 专注的表情
  - 手部聚集能量光球
  - 衣物被魔力吹动
  - 强烈的光效
```

**3. descent_pose_casting.png（施法姿势）**
```
描述：叙事者全力施展降临魔法的瞬间
视觉要素：
  - 凌厉的表情
  - 手部释放能量
  - 强烈的光效爆发
  - 魔法阵浮现（可选）
```

#### 分层立绘（可选）
如果需要动态效果，可以将姿势拆分为多层：
- **body.png**: 身体基础层
- **effects_glow.png**: 光效层（可动画）
- **effects_particles.png**: 粒子层（可动画）

### 2. 降临特效（Effects/）

#### 援助模式特效（Assist/）

**视觉风格**：温暖、神圣、治愈
- **色调**: 金色、白色、浅蓝色
- **氛围**: 光明、希望、保护

**特效列表**：

| 文件名 | 用途 | 尺寸 | 描述 |
|--------|------|------|------|
| aura_healing_01.png | 治愈光环 | 512x512 | 柔和的金色光晕，向外扩散 |
| aura_blessing_01.png | 祝福光环 | 512x512 | 闪烁的白色星光环绕 |
| particles_sparkle.png | 星光粒子 | 256x256 | 漂浮的小星星粒子 |
| ground_circle_holy.png | 神圣法阵 | 1024x1024 | 地面上的金色魔法阵 |

**动画建议**：
- 光环：旋转 + 脉冲缩放（1.0 → 1.2 → 1.0）
- 粒子：向上漂浮 + 淡入淡出
- 法阵：旋转 + 发光强度变化

#### 袭击模式特效（Attack/）

**视觉风格**：黑暗、威严、毁灭
- **色调**: 红色、黑色、紫色
- **氛围**: 恐惧、力量、压迫

**特效列表**：

| 文件名 | 用途 | 尺寸 | 描述 |
|--------|------|------|------|

**动画建议**：
- 光环：剧烈抖动 + 不规则缩放

### 3. 入场过场动画（Cinematic/）

#### 龙骑兵入场动画（DragonRider/）

**动画序列设计**：

```
总时长：6-8 秒
帧率：15 FPS（总共 90-120 帧）
分辨率：1920x1080（全屏）或 1280x720（窗口）
```

**序列分段**：

**阶段 1：接近（Approach）** - 帧 1-40
```
场景：天空
镜头：远景 → 近景
动作：龙骑兵从天际线飞来，逐渐放大
特效：云雾穿行、翅膀扇动、风流效果
```

**阶段 2：盘旋（Circle）** - 帧 41-60
```
场景：战场上空
镜头：跟随镜头
动作：巨龙盘旋，寻找降落点
特效：俯瞰地面、阴影投射
```

**阶段 3：降落（Landing）** - 帧 61-90
```
场景：降落地点
镜头：地面仰视
动作：巨龙俯冲降落，产生冲击波
特效：地面龟裂、烟尘四起、能量爆发
```

**阶段 4：下马（Dismount）** - 帧 91-120
```
场景：降落点特写
镜头：侧面中景
动作：叙事者从龙背跳下，摆出姿势
特效：光环浮现、武器显现（可选）
```

**序列帧命名规范**：
```
rider_approach_001.png  ~ rider_approach_040.png
rider_circle_041.png    ~ rider_circle_060.png
rider_landing_061.png   ~ rider_landing_090.png
rider_dismount_091.png  ~ rider_dismount_120.png
```

#### 可选入场方式

**传送门入场（Portal/）**
```
效果：空间裂缝打开，叙事者从中走出
时长：4-5 秒
特点：更快捷，更神秘
```

**瞬移入场（Teleport/）**
```
效果：光柱降下，叙事者凭空出现
时长：2-3 秒
特点：最简洁，适合紧急情况
```

### 4. 降临后小人图像（Pawns/）

#### 规格要求
- **尺寸**: 128x128 像素（标准小人尺寸）
- **格式**: PNG（带 Alpha 透明通道）
- **分辨率**: 72 DPI（游戏内资源）
- **视角**: 45° 斜视角（RimWorld 标准）

#### 三视图规范

**front.png（正面图）**
```
用途：小人面向下方时显示
要求：
  - 清晰的面部特征
  - 完整的装备细节
  - 自然的站立姿势
  - 阴影适中
```

**side.png（侧面图）**
```
用途：小人面向左/右时显示
要求：
  - 侧脸轮廓清晰
  - 装备厚度合理
  - 自然的侧面姿态
  - 与正面图风格一致
```

**back.png（背面图）**
```
用途：小人面向上方时显示
要求：
  - 背部装备完整
  - 头发/头盔背面细节
  - 与正背面对称
  - 阴影方向一致
```

**portrait.png（头像）**
```
用途：UI界面头像显示
要求：
  - 正面半身像
  - 高辨识度
  - 可用于对话框
```

#### 装备与状态变体（可选）

可以为不同状态创建变体：
```
front_naked.png      # 裸体基础图（用于穿衣服系统）
front_armed.png      # 持武器状态
front_damaged.png    # 受伤状态
front_downed.png     # 倒地状态
```

## ?? 技术规范

### 纹理压缩

**推荐设置**：
- **压缩格式**: DXT5（RimWorld 标准）
- **Mipmap**: 启用（远距离清晰）
- **Filter Mode**: Bilinear（平滑）
- **Wrap Mode**: Clamp（避免边缘重复）

### 透明度处理

**Alpha 通道规范**：
```
完全透明：Alpha = 0（背景）
半透明：  Alpha = 128（光效、粒子）
完全不透明：Alpha = 255（实体部分）
```

### 文件命名规范

**统一格式**：
```
[类型]_[名称]_[编号].png

示例：
aura_healing_01.png        # 特效光环
rider_landing_061.png      # 动画序列帧
dragon_front.png           # 生物三视图
```
