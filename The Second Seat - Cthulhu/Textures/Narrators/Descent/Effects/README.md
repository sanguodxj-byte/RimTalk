# Cthulhu 降临特效

克苏鲁娘的降临特效资源。

## ?? 文件放置

```
Textures/Narrators/Descent/Effects/Cthulhu/
├── effect_assist.png  ← 援助特效（扭曲守护）
└── effect_attack.png  ← 袭击特效（疯狂侵蚀）
```

## ?? Cthulhu 特效风格

### 援助特效（effect_assist.png）
- **主题**: 深海守护、混沌秩序
- **色调**: 深绿色 + 紫色 + 青色
- **元素**: 
  - 触手纹理
  - 水波涟漪
  - 深海气泡
  - 神秘符文

### 袭击特效（effect_attack.png）
- **主题**: 疯狂侵蚀、理智崩溃
- **色调**: 暗红色 + 黑紫色
- **元素**:
  - 扭曲的触手
  - 破碎的现实
  - 疯狂的符号
  - 精神冲击波

## ?? 制作参考

### 援助特效灵感
- 深海生物发光
- 水下氛围（半透明）
- 克苏鲁神话符号
- 异界守护结界

### 袭击特效灵感
- 恐怖游戏视觉效果
- 精神污染特效
- 血肉扭曲
- SAN值下降视觉

## ?? Def 配置

在 `Defs/NarratorPersonaDefs_Cthulhu.xml` 中：

```xml
<descentEffectPath>effect</descentEffectPath>
```

## ?? 特别注意

### 疯狂机制联动
- 袭击特效应暗示「疯狂」主题
- 可以加入：
  - 眼睛符号
  - 扭曲的几何
  - 不规则的纹理

### 色彩选择
- **避免**: 过于鲜艳的颜色
- **推荐**: 深色调、低饱和度
- **重点**: 营造压迫感和神秘感

## ?? 规格标准

- **尺寸**: 1024x2048px
- **格式**: PNG-24（RGBA）
- **文件大小**: < 2MB
- **命名**: `effect_assist.png`, `effect_attack.png`

## ?? 克系美术风格指南

1. **触手元素**: 流动感、有机形态
2. **深海色调**: 暗绿、暗蓝、紫黑
3. **神秘符文**: 抽象、不对称
4. **疯狂暗示**: 扭曲、破碎、多重
