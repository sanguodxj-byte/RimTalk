克苏鲁娘的降临姿态资源。

```

```

## ?? 姿态设计指南

### 降临姿势特征
- **姿态**: 从天而降/海中升起
- **动作**: 触手展开、双臂伸展
- **表情**: 邪魅/冷漠/神秘
- **细节**: 
  - 触手自然延伸
  - 衣物飘动感
  - 深海气息

### 克苏鲁娘特色元素
- **触手**: 从身体或衣物延伸
- **深海**: 海藻、气泡、水流
- **神秘**: 发光的眼睛/符文
- **混沌**: 不规则的阴影

## ?? 构图建议

### 视觉重心
- 头部位于上 1/3
- 身体微微倾斜（动态感）
- 触手填充空白区域

### 动态感营造
- 头发向上飘
- 触手向外伸展
- 衣物飘动
- 碎片/气泡环绕

## ?? 规格要求

- **尺寸**: 1024x2048px
- **格式**: PNG-24（RGBA）
- **背景**: 透明或半透明
- **文件大小**: < 3MB

## ?? 克系姿态参考

### 降临姿势灵感
- 《炉石传说》尼奥罗萨
- 触手系生物降临动作

- 异界之门开启

### 触手绘制技巧
1. **基础形态**: 先画主干
2. **吸盘细节**: 间隔分布
3. **明暗变化**: 体现立体感
4. **动态线条**: 流畅自然

## ?? 注意事项

### 必须避免
- ? 过于血腥/恐怖

- ? 过于复杂的细节

### 推荐做法

- ? 优雅的触手
- ? 保留美感
