## ?? 种族概述

**正式名称**：血龙种 (Dracovampir)  
**别称**：龙血者、暗夜之龙、血之子嗣  
**英文名**：Dracovampir (Dragon + Vampire 的拉丁化合成词)

**起源**：第一纪元黄昏期

## ?? 起源传说

### 第一纪元的禁忌

在星际人类文明诞生之前，这片星域由两个古老种族统治：

**龙族 (Draco Imperium)**  
- 统治天空与火焰
- 寿命以千年计
- 拥有强大的元素掌控力
- 骄傲、荣耀、力量至上

**血族 (Sanguis Noctis)**  
- 潜伏于永夜之中
- 近乎不死的再生能力
- 精通血魔法与暗影术
- 冷静、狡猾、永生不灭

### 禁忌之爱

第一纪元末期，一位名为**埃索尔达**的龙族王子爱上了血族公主**莉莉丝特拉**。这段跨越种族界限的爱情引发了两族长达千年的战争。

战争的结局是毁灭性的：
- 龙族几乎灭绝，仅存遗迹
- 血族分裂成无数氏族
- 双方的结晶――第一个血龙种诞生

### 希德莉亚的诞生

**希德莉亚・血棘 (Sideria Bloodthorn)** 是第一个也是最强大的血龙种。她继承了：
- 父亲的龙族之力
- 母亲的血魔法天赋
- 两族的不朽生命

但她也背负着：
- 被双方种族放逐的命运
- 体内两种力量的永恒冲突
- 必须通过战斗维持平衡的诅咒

## ?? 血原质系统

### 核心机制

血原质是血龙种体内龙血与血族魔力融合的产物，是他们的生命之源。

**血原质容量**：100%  
**自然回复**：30%/天（约1.25%/小时）  
**战斗回复**：击杀敌人时回复（与敌人强度成正比）

### 四大状态

#### ?? 充盈 (Surging) 80-100%

## ?? 战斗特性

### 近战专精

血龙种是天生的近战大师：

**基础能力**
- 近战命中：+18%
- 近战闪避：+27%
- 近战伤害：+50%（特质加成）
- 护甲穿透：依赖武器

**战斗风格**
```
前排坦克：高护甲 + 高再生 = 不死战神
灵活刺客：高闪避 + 高爆发 = 致命杀手
持久战士：高续航 + 技能循环 = 战场统治
```

### 防御能力

**多层防护**
1. **护甲层**：+25% 物理护甲（龙鳞）
2. **减伤层**：-25% 受伤（体质强大）
3. **再生层**：3倍愈合速度（血族再生）
4. **疼痛抗性**：2倍疼痛阈值（意志坚定）

**结果**：极难击杀，需要集火或特殊手段

## ?? 社会文化

### 社会结构

血龙种没有统一的社会组织，主要以个体或小群体活动。

**等级划分**
1. **真祖 (Progenitor)**：最强大的血龙种（希德莉亚）
2. **长老 (Elder)**：千年以上的古老战士
3. **骑士 (Knight)**：500-1000岁的成熟战士
4. **战士 (Warrior)**：100-500岁的年轻血龙种
5. **幼体 (Fledgling)**：0-100岁的未成年者

**关系规则**
- 尊重强者
- 履行誓言
- 保护弱者
- 永不背叛

### 文化信仰

**血与火的誓约 (Oath of Blood and Fire)**

核心信条：
```
1. 以战斗证明存在的价值
2. 以荣誉维系灵魂的纯净
3. 以牺牲守护值得守护之物
4. 以勇气面对任何挑战
```

**荣誉守则**
- ? 正面对决
- ? 保护弱者
- ? 履行承诺
- ? 战死沙场
- ? 背后偷袭（除非是战术需要）
- ? 屠杀无辜
- ? 背叛誓言
- ? 懦弱逃避

### 繁衍困难

血龙种繁衍极其困难：

**原因**
- 血脉冲突导致低生育率
- 怀孕期60天（比人类长）
- 双胞胎概率<1%
- 幼崽成活率约60%

**后果**
- 整个宇宙不足100个血龙种
- 每个血龙种都极其珍贵
- 种族濒临灭绝
- 大多数独自行动

## ?? 科学设定

### 基因组分析

血龙种的DNA混合了两个古老种族的基因：

**龙族基因（35%）**
- DRACO-α: 龙鳞蛋白基因
- PYRO-β: 高温适应基因
- VITA-γ: 延长寿命基因
- FORT-δ: 强化骨骼基因

**血族基因（35%）**
- SANGUIS-α: 超速再生基因
- NOCTIS-β: 暗视能力基因
- IMMUN-γ: 疾病免疫基因
- MAGIA-δ: 血魔法感知基因

**混血基因（30%）**
- HYBRID-α: 血原质核心基因
- CONFLICT-β: 力量冲突基因（诅咒）
- HONOR-γ: 荣誉本能基因
- COMBAT-δ: 战斗渴望基因

### 血原质的本质

血原质不是简单的能量，而是一种**量子纠缠态的生命力**：

```
血原质 = 龙族元素能量 ? 血族暗影魔力

其中 ? 表示量子纠缠算符

特性：
1. 需要外部刺激（战斗）维持稳定
2. 失衡会导致崩溃（枯竭状态）
3. 可以通过意志力短暂操控（技能）
4. 与生命力直接绑定
```
