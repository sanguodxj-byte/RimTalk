# 纹理文件说明

## 图标位置

`Textures/UI/Commands/ListenToMe.png`

## 要求

- **格式**: PNG
- **尺寸**: 64x64 像素（推荐）或 38x38 像素
- **背景**: 透明
- **风格**: 符合 RimWorld 美术风格

## 设计建议

### 推荐图标元素
- ?? 对话气泡
- ?? 文本/字母
- ?? 指令箭头
- ??? 麦克风
- ?? 手写笔

### 颜色建议
- 主色: 白色或浅灰色
- 强调色: 蓝色或绿色
- 避免使用: 红色（表示危险）

## 临时解决方案

当前代码已配置为在没有自定义图标时使用游戏内置图标：
- 默认使用: `TexCommand.Draft`（征召图标）

## 创建自定义图标

### 方法1: 使用图像编辑器
1. 打开 Photoshop / GIMP / Paint.NET
2. 创建 64x64 像素画布
3. 背景设为透明
4. 设计图标
5. 导出为 PNG
6. 保存到 `Textures/UI/Commands/ListenToMe.png`

### 方法2: 使用在线工具
1. 访问在线图标编辑器（如 Pixlr）
2. 创建 64x64 图标
3. 下载为 PNG
4. 放置到正确位置

### 方法3: 使用现有资源
可以从 RimWorld 游戏文件中提取现有图标作为参考：
```
RimWorld\Data\Core\Textures\UI\Commands\
```

## 测试图标

1. 将图标放置到 `Textures/UI/Commands/ListenToMe.png`
2. 重启 RimWorld
3. 选中小人查看按钮
4. 图标应该显示
