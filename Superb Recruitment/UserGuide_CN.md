# Superb Recruitment 使用指南

## 快速开始

### 1. 基本流程

```
访客到来 → 选中访客 → 点击"说服"按钮 → 选择说服者 → 进行对话 → 重复对话提升说服值 → 尝试招募
```

### 2. 说服机制

#### 说服值系统
- **初始值**: 每个访客的说服值在20%-80%之间随机生成（正态分布，中心50%）
- **隐藏性**: 玩家看不到具体数值，只能看到"?"
- **范围**: 0% - 99%（永远不会达到100%）

#### 尝试次数限制
- **玩家**: 可以亲自说服同一访客**3次**
- **殖民者**: 每个殖民者只能说服同一访客**1次**

### 3. 说服效果

#### 玩家说服 (对话窗口)
根据选择的对话选项：
- **友好闲聊**: 5%-12% 提升（难度：简单）
- **分享故事**: 8%-15% 提升（难度：中等）
- **描述机会**: 10%-20% 提升（难度：困难）
- **情感诉求**: 0%-25% 提升（难度：非常困难）
- **简短交谈**: 2%-5% 提升（难度：非常简单）

**注意**: 高难度选项成功时效果好，失败时效果差

#### NPC殖民者说服
- **基础效果**: 8%
- **社交技能**: 每级 +0.5%
- **善良特性**: +15%
- **精神病质**: -20%
- **随机变化**: ±20%

### 4. 招募机制

#### 何时可以招募
当说服值足够高时（成功率 ≥ 10%），会出现"尝试招募"按钮

#### 成功率计算
```
成功率 = (说服值)^2.5
```

例如：
- 说服值30% → 成功率约4.9%
- 说服值50% → 成功率约17.7%
- 说服值70% → 成功率约42.7%
- 说服值90% → 成功率约72.5%

#### 招募结果
- **成功**: 访客立即加入殖民地，获得正面情绪
- **失败**: 说服值降低15%，访客获得轻微负面情绪
- **过度失败**: 如果说服值降到10%以下，访客会生气离开

## 进阶技巧

### 1. 选择合适的说服者

优先选择：
- **高社交技能**的殖民者
- 拥有**善良**特性的殖民者
- 避免使用有**精神病质**的殖民者

### 2. 策略规划

#### 保守策略
1. 先用殖民者说服2-3次
2. 再用玩家尝试1-2次
3. 等说服值较高时再尝试招募

优点：稳妥，浪费的尝试次数少
缺点：可能需要更多访客才能成功

#### 激进策略
1. 玩家选择高难度对话选项
2. 快速提升说服值
3. 尽早尝试招募

优点：有可能快速成功
缺点：风险高，可能导致访客离开

### 3. 理解概率

重要提示：
- 显示的成功率只是**估算值**
- 实际成功率是隐藏的
- 多次失败不会提高下次成功率（每次独立）

### 4. 何时放弃

以下情况建议停止招募：
- 说服值低于15%
- 已经失败3次以上
- 访客即将离开

## 常见问题

### Q: 为什么我的殖民者无法说服？
A: 可能原因：
1. 该殖民者已经尝试过了
2. 殖民者倒下/睡觉/死亡
3. 殖民者与访客敌对

### Q: 说服值会自然下降吗？
A: 不会。说服值只在以下情况变化：
- 成功说服：上升
- 招募失败：下降15%

### Q: 可以对囚犯使用吗？
A: 不可以。这个系统只针对**访客**（Guest status）

### Q: 如果访客离开了怎么办？
A: 说服进度会保存在访客身上。如果同一个访客再次来访，可以继续说服。

### Q: RimTalk 是必需的吗？
A: 不是。没有 RimTalk 时会使用内置的简单对话窗口。

## 游戏平衡建议

### 难度调整

如果你觉得太难/太容易，可以修改源代码：

#### 增加成功率
在 `Hediff_PersuasionTracking.cs` 中修改：
```csharp
// 从 2.5 改为 2.0
float successChance = (float)Math.Pow(persuasionValue, 2.0);
```

#### 增加尝试次数
在 `Hediff_PersuasionTracking.cs` 中修改：
```csharp
private int playerAttemptsRemaining = 5; // 从 3 改为 5
private int colonistAttemptsRemaining = 2; // 从 1 改为 2
```

#### 增加说服效果
在 `PersuasionDialogueManager.cs` 中修改：
```csharp
float baseEffect = 0.12f; // 从 0.08f 提高到 0.12f
```
